=head1 NAME

Web::DOM::HTMLFrameSetElement - The DOM |HTMLFrameSetElement| object

=head1 DESCRIPTION

The C<Web::DOM::HTMLFrameSetElement> class implements the DOM
C<HTMLFrameSetElement> interface.  The class is used by the HTML
C<frameset> element.

The interface, as well as the C<frameset> element, is obsolete.

=head1 METHODS

In addition to the methods available for classes L<Web::DOM::Element>
and L<Web::DOM::HTMLElement>, the following obsolete methods are
available: C<cols> and C<rows>.  [HTML]

=head1 SPECIFICATION

=over 4

=item HTML

HTML Standard
<http://www.whatwg.org/specs/web-apps/current-work/#the-frames-element>.
