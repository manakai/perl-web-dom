=head1 NAME

Web::DOM::Attr - The DOM |Attr| object

=head1 DESCRIPTION

The C<Web::DOM::Attr> class implements the DOM C<Attr> interface.

=head1 METHODS

The methods of the L<Web::DOM::Node> class are available.
