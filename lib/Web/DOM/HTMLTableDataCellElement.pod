=head1 NAME

Web::DOM::HTMLTableDataCellElement - The DOM |HTMLTableDataCellElement| object

=head1 DESCRIPTION

The C<Web::DOM::HTMLTableDataCellElement> class implements the DOM
C<HTMLTableDataCellElement> interface.  The class is used by the HTML
C<td> element.

=head1 METHODS

In addition to the methods available for classes L<Web::DOM::Element>,
L<Web::DOM::HTMLElement>, and L<Web::DOM::HTMLTableCellElement>, the
obsolete C<abbr> method is available.  [HTML]

=head1 SPECIFICATION

=over 4

=item HTML

HTML Standard
<http://www.whatwg.org/specs/web-apps/current-work/#the-td-element>.

=back
