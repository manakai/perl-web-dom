=head1 NAME

Web::DOM::HTMLHeadingElement - The DOM |HTMLHeadingElement| object

=head1 DESCRIPTION

The C<Web::DOM::HTMLHeadingElement> class implements the DOM
C<HTMLHeadingElement> interface.  The class is used by the HTML C<h1>,
C<h2>, C<h3>, C<h4>, C<h5>, and C<h6> elements.
