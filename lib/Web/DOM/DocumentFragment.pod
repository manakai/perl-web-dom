=head1 NAME

Web::DOM::DocumentFragment - The DOM |DocumentFragment| object

=head1 DESCRIPTION

The C<Web::DOM::DocumentFragment> class implements the DOM
C<DocumentFragment> interface.

=head1 METHODS

The methods of classes L<Web::DOM::EventTarget>, L<Web::DOM::Node>,
and L<Web::DOM::ParentNode> are available.

=head1 SPECIFICATION

=over 4

=item DOM

DOM Living Standard
<http://dom.spec.whatwg.org/#interface-documentfragment>.

=back
