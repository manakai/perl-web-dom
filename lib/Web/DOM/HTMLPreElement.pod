=head1 NAME

Web::DOM::HTMLPreElement - DOM |HTMLPreElement| object

=head1 DESCRIPTION

The C<Web::DOM::HTMLPreElement> class implements the DOM
C<HTMLPreElement> interface.  The class is used by the HTML C<pre>
element (and C<listing> and C<pre> elements).

=head1 METHODS

In addition to the methods available for classes L<Web::DOM::Element>
and L<Web::DOM::HTMLElement>, the obsolete C<width> method is
supported. [HTML]

=back

=head1 SPECIFICATION

=over 4

=item HTML

HTML Standard
<https://www.whatwg.org/specs/web-apps/current-work/#the-pre-element>.

=back
