=head1 NAME

Web::DOM::AtomUriElement - The DOM |AtomUriElement| object

=head1 DESCRIPTION

The C<Web::DOM::AtomUriElement> class implements the DOM
C<AtomUriElement> interface.  The class is used by the Atom C<uri>
element.

=head1 METHODS

The methods for classes L<Web::DOM::Element> and
L<Web::DOM::AtomElement> are available.

=head1 SPECIFICATION

=over 4

=item ATOMDOM

Atom DOM <http://suika.suikawiki.org/www/markup/xml/atomdom>.

=back
