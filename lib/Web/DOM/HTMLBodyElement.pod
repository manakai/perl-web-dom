=head1 NAME

Web::DOM::HTMLBodyElement - The DOM |HTMLBodyElement| object

=head1 DESCRIPTION

The C<Web::DOM::HTMLBodyElement> class implements the DOM
C<HTMLBodyElement> interface.  The class is used by the HTML C<body>
element.

=head1 METHODS

In addition to the methods available for classes L<Web::DOM::Element>
and L<Web::DOM::HTMLElement>, the following obsolete methods are
available: C<alink>, C<background>, C<bgcolor>, C<text>, C<link>, and
C<vlink>.  [HTML]

=head1 SPECIFICATION

=over 4

=item HTML

HTML Standard
<http://www.whatwg.org/specs/web-apps/current-work/#the-body-element>.

=back
