=head1 NAME

Web::DOM::HTMLHRElement - The DOM |HTMLHRElement| object

=head1 DESCRIPTION

The C<Web::DOM::HTMLHRElement> class implements the DOM
C<HTMLHRElement> interface.  The class is used by the HTML C<hr>
element.

=head1 METHODS

In addition to the methods available for classes L<Web::DOM::Element>
and L<Web::DOM::HTMLElement>, the following obsolete methods are
available: C<align>, C<color>, C<noshade>, c<size>, and C<width>.
[HTML]

=head1 SPECIFICATION

=over 4

=item HTML

HTML Standard
<http://www.whatwg.org/specs/web-apps/current-work/#the-hr-element>.

=back
