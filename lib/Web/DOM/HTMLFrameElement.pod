=head1 NAME

Web::DOM::HTMLFrameElement - The DOM |HTMLFrameElement| object

=head1 DESCRIPTION

The C<Web::DOM::HTMLFrameElement> class implements the DOM
C<HTMLFrameElement> interface.  The class is used by the HTML C<frame>
element.

The interface, as well as the C<frame> element, is obsolete.

=head1 METHODS

In addition to the methods available for classes L<Web::DOM::Element>
and L<Web::DOM::HTMLElement>, the following obsolete methods are
available: C<frameborder>, C<longdesc>, C<marginheight>,
C<marginwidth>, C<name>, C<noresize>, C<scrolling>, and C<src>.
[HTML]

=back

<http://www.whatwg.org/specs/web-apps/current-work/#the-frames-element>.
